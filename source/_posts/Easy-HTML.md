## HTML



### Basic Elements

1. `<body>`: show in windows
2. `<head>`: page info
3. `<title>`: tab title



## Text

`<h [1-6]>`: title

`<p>`: paragraph

`<b>`: bold

`<i>`: italic 

`<sup>`: superscript 

`<sub>`: subscript

`<br />`:  break line

`<hr>`: horizontal line

`<em>`: inline element(strong emphasized)

`<blockquote>`: quote

`<q>`: short quote

`<abbr title=“whole word”>ABB</abbr>`: abbreviation

`<cite>`: citation to book or film

`<dfn>`: definition of something 

`<address>`: include the info of author

`<ins>`: insert(underline)

`<del>`: deleted line

`<s>`:  deleted line



### List

`<ol>`: ordered list

`<li>`: list item

`<ul>`: unordered  list

`<dl>`: definition list

`<dt>`: definition term

`<dd>`:definition

lists can be nested.
